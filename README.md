# Paradise Nursery Shopping Cart

## Project Name
Paradise Nursery - E-Commerce Plant Shopping Application

## Description
A React-based e-commerce application for purchasing plants. Features include browsing different plant categories, adding items to cart, managing quantities, and viewing total costs.

## Features
- Browse plants by category (Air Purifying, Aromatic, Insect Repellent, Medicinal, Low Maintenance)
- Add plants to shopping cart
- Increment/decrement item quantities
- Remove items from cart
- Real-time cart total calculation
- Redux state management
- Responsive design

## Technologies Used
- React
- Redux Toolkit
- Vite
- CSS
